Project 1

This is the first project of the CIS 3342 - Server-Side Web Application Development. In this project, I learned how to create a web application with a simple HTML page with a form processed by the ASPX server-side page that respond to the request from the HTML page.

This first project was a useful guide to help me starting undestanding how Client-side and Server-side web application communicate to each other. Overall, the project was simple once I fully understand the concept and how to process it, though it was a bit confusing at first, but by reviewing materials and reading from the book, it helps me understand so much better and I was able to complete the project. On the other hand, since I am not taking Client-Side, CIS 3344, I had some hard time doing my own research on CSS to make the client page look presentable and clear to the user.

Project 2

The 2nd project gives me experience with creating web applications by using GridView control and dynamic binding. I designed a Book Store in this project, list all book options in a GridView for user to pick and make the order. In this project, I only need to create one single ASPX webpage, and learn to create a postback webpage, which the entire page and its content are sent to the server for processing information and then the server posts the same page back to the browser.

This is a good project and I learned a lot from it. I had less trouble with this project than the first one, as I got more familar and I used the same CSS layout from project 1 for this one.

Project 3
